# Verification 2.0


## Technologies used

- **Monorepo Tooling:** Turborepo  
- **Frontend:** React, Vite, Tailwind CSS, Vitest
- **Backend:** Node.js, Express

## Setup & Installation
1. Install dependencies

```bash
pnpm install
```

2. Start development servers (both frontend and backend)

```bash
pnpm dev
```

## Running tests

```bash
pnpm test
```
